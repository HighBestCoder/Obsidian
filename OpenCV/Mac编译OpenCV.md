需要修改一下系统目录

```
#!/bin/bash

# 指定 opencv 目录

# 遍历 opencv 目录下的所有文件
find "." -type f -print0 | while IFS= read -r -d '' file; do
    # 替换文件中的文本
    sed -i '' 's/MacOSX.sdk/MacOSX.sdk/g' "$file"
    echo "Updated: $file"
done
```
